RugSonar API

Embed token risk scanning in your bot, wallet, or dashboard. One request returns a 0–100 score, the evidence behind it, and a plain-English explanation of what the contract owner can do to a holder.

Every other risk API hands your users 30 cryptic boolean flags. RugSonar aggregates five security sources into a single deterministic score and explains it — the one thing none of the incumbents ship in an API.

Base URL

https://rugsonar.com

Authentication

Send your key as a bearer token. Without a key you share the public rate limit; a key gives you a metered monthly quota and commercial use.

Authorization: Bearer rsk_live_xxxxxxxxxxxxxxxx

Endpoints

GET /v1/scan/:chain/:address GET /v1/scan?address= GET /v1/usage

Chains: eth bsc base polygon arbitrum optimism avalanche solana tron. Omit the chain and pass ?address= to auto-detect from the address format.

Example

curl https://rugsonar.com/v1/scan/eth/0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2 \
     -H "Authorization: Bearer rsk_live_xxxxxxxxxxxxxxxx"
{
  "chain": "eth",
  "chainName": "Ethereum",
  "address": "0xc02aaa39b223fe8d0a0e5c4f27ead9083c756cc2",
  "score": 85,
  "verdict": "no-red-flags",
  "findings": [
    { "severity": "low", "title": "Single wallet holds outsized supply",
      "evidence": "Top holder controls 24% of supply", "source": "goplus" }
  ],
  "facts": { "name": "Wrapped Ether", "symbol": "WETH", "sourceVerified": true, "...": "..." },
  "checks": [ { "source": "etherscan", "ok": true }, "..." ],
  "partial": false,
  "aiExplanation": "WETH is a wrapped version of native Ether with no red flags...",
  "scannedAt": "2026-07-06T06:20:00.000Z"
}

Verdicts: high-risk, risky, caution, no-red-flags. The score is deterministic — the AI only explains the findings, it never changes the number. On unverified source the explanation is omitted.

Usage & limits

Check your remaining quota any time:

curl https://rugsonar.com/v1/usage -H "Authorization: Bearer rsk_live_..."
{ "tier": "indie", "month": "2026-07", "used": 3410, "limit": 25000, "remaining": 21590 }

Over quota returns 429 with the tier and limit. Quotas reset on the 1st (UTC).
